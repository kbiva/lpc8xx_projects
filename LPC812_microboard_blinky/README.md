Introduction
=
LPC812 microboard and one blinking LED.

Hardware
=
* LPC812 microboard. (MCU clock is 12Mhz from internal oscillator).
* One LED.

Software
=
Used peripherals: 

* GPIO.

Project uses external libraries:

* LPCOpen 3.02 library for LPC8xx
